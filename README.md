## 📰 Fake News Detection App

This is a web-based application to detect whether a news article is **Fake** or **Real** using Machine Learning.

## 🔧 Technologies Used

- Python
- Flask
- Scikit-learn
- NLTK
- HTML (Bootstrap)

## 🚀 How to Run
